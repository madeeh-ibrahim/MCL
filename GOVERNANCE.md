# Governance

MCL is currently a **solo-maintainer project**, owned and maintained by
Madeeh Ibrahim (ORCID https://orcid.org/0009-0002-8562-8325). All technical,
licensing, and release decisions rest with the maintainer.

External contributions are welcome under the MCL Individual CLA (see `CLA.md`).
The maintainer commits to:
- acknowledging non-trivial contributions in release notes and `CITATION.cff`;
- reviewing security reports within the targets in `SECURITY.md`;
- maintaining a transparent log of cryptanalytic findings in `HALL_OF_FAME.md`.

If the project transitions to multi-maintainer governance, this file will be
updated and a corporate CLA introduced. Until then, the maintainer reserves all
rights not expressly delegated.
