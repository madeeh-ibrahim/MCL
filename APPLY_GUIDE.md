the audited strategy. It integrates all eight fixes from the deep audit. Below:
what each file is, what you must fill in, and the order of operations.

## 2. How the 8 audit fixes were applied

1. **SPDX = PolyForm only (not `AND`/`OR`)** → every header, `REUSE.toml`, and
   `add_spdx_headers.sh` use `PolyForm-Noncommercial-1.0.0` alone; the Grant is
   documented in prose. Rationale noted inline in `REUSE.toml` and the script.
2. **CITATION.cff license fixed** → `license: PolyForm-Noncommercial-1.0.0`
   (no `LicenseRef-` prefix); the Grant is described in `message`.
3. **§287 → §154(d)** → `PATENTS.md` frames the notice as "patent pending" +
   §154(d) provisional rights now; §287 virtual marking only "once a patent
   issues."
4. **Jurisdiction-neutral safe harbor** → `SECURITY.md` removes CFAA/DMCA and
   uses disclose.io-style neutral wording.
5. **Cover-note presents both instruments** → top of `LICENSE` (and echoed in
   `NOTICE` and the `README` license table) so commercial researchers see the
   Grant immediately.
6. **COMMERCIAL.md = simple referral** → the detailed V3.6 arbitration/venue
   agreement stays a PRIVATE signed contract, not published here.
7. **Retaliation harmonized** → `PATENTS.md`, `SECURITY-RESEARCH-GRANT.md` §5,
   and `CLA.md` §3 all use the same narrow trigger ("a claim that THIS software
   infringes"), aligned to PolyForm's Patent Defense.
8. **README alternatives tailored by category** → README recommends category-
   appropriate primitives (AEAD, MAC, DRBG, PQC) rather than a mismatched list.
   (Note: no static CC0 test-vector set is shipped; this project generates KAT
   data via kat_gen_macos.cpp, so the CC0 license file was removed.)

## 4. Adding SPDX headers to your code

```bash
# Copy the script next to your source files, then:
bash add_spdx_headers.sh /path/to/your/mcl/source
# It puts the FULL header on mcl_core.hpp and the SHORT header on every other
# .cpp/.hpp/.h/.cc/.py file. It is idempotent (skips files already tagged) and
# preserves Python shebang lines.
```

Optionally verify with REUSE:

```bash
pipx run reuse lint
```
